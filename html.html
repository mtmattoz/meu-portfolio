O que é HTML?

HTML (HyperText Markup Language) é a linguagem padrão para criação e estruturação de páginas web. Ele define a estrutura básica e o conteúdo de uma página web, incluindo texto, imagens, vídeos e links.

Estrutura Básica do HTML

Um documento HTML básico consiste em elementos fundamentais, como <!DOCTYPE html>, <html>, <head> e <body>.

            
                <!DOCTYPE html>
                <html lang="pt-BR">
                <head>
                    <meta charset="UTF-8">
                    <title>Título da Página</title>
                </head>
                <body>
                    <!-- Conteúdo da Página -->
                </body>
                </html>

Tags e Elementos do HTML

O HTML é composto por tags que são utilizadas para marcar diferentes partes do conteúdo de uma página web. Algumas tags comuns incluem:

  • <h1> - <h6>: Títulos e subtítulos
  • <p>: Parágrafos de texto
  • <a>: Links
  • <img>: Imagens
  • <ul> e <ol>: Listas não ordenadas e ordenadas
  • <div>: Divisões para agrupamento de elementos
